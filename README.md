## 🎨 Project 1: Responsive Frontend Interface (The Skin)
A pure, framework-less frontend built following the "Mobile-First" paradigm and strict 2025 UI/UX Compliance.

## ✨ Key Features
- Zero Frameworks: Built entirely using Pure HTML5, CSS3, and Vanilla JavaScript.
- Mobile-First Architecture: Responsive design scaling smoothly from mobile to desktop using CSS Grid (Macro layouts) and Flexbox (Micro components).
- Semantic Integrity: Strict adherence to semantic HTML5 (<header>, <main>, <article>, etc.) for optimal AI readability and WCAG accessibility.
- 2025 Aesthetics: Utilizes the official palette (Mocha Mousse, Ethereal Blue, Moonlit Grey) with fluid typography (CSS clamp()).

## ⚙️ Project 2: Backend API Development (The Nervous System)
A stateless, secure, and robust backend API designed to handle the application's core logic with absolute reliability.

## 🛡️ Key Features
- RESTful Architecture: Clean endpoint naming conventions (Resources as Nouns, Methods as Verbs).
- The Gatekeeper Rule: "Never Trust the Client". Implements robust Syntactic (format) and Semantic (logic) validation before data touches the database.
- Precise State Communication: Strict usage of semantic HTTP status codes (201 Created, 204 No Content, 400 Bad Request, 401 Unauthorized, 500 Internal Error, etc.).
- JSON Data Exchange: Lightweight, machine-parsable, and human-readable data flow.

## 🚀 Getting Started
Prerequisites
Node.js or Python installed on your machine.
A modern web browser.

Installation & Setup
1. Clone the repository:

Bash

```
git clone https://github.com/ydagunawardhana/DecodeLabs-Internship
cd DecodeLabs-Internship
```
2. Run the Frontend (Project 1):

- Navigate to the frontend/ folder.
- Open index.html in your browser (or use a tool like VS Code Live Server).

3. Run the Backend (Project 2):

- Navigate to the backend/ folder.
- Install dependencies (e.g., npm install).
- Start the server (e.g., npm start or node server.js).
